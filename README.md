# 简易记账本

一个纯前端、离线可用的个人记账工具。

## 功能
- 按类别（收入/支出）和项目分类记账
- 支持农行私账、备用金两种支付方式
- 自动生成月度凭证号（001, 002...）
- 按支付方式分别统计余额
- CSV 批量导入/导出
- 数据仅保存在本地浏览器，隐私安全

## 使用
1. 访问 [https://yourname.github.io/accounting-app/](https://yourname.github.io/accounting-app/)
2. 直接开始记账
3. 关闭后数据仍保留（基于 localStorage）

> 💡 无需登录，无需网络，所有数据只存在你自己的设备上。
